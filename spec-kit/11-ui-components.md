# 11 — قائمة مكونات واجهة المستخدم (UI Component Inventory)

## 1. مكونات التنقل (Navigation)

### 1.1 Navbar — شريط التنقل الرئيسي
- **الوصف:** شريط تنقل أفقي ثابت أعلى الصفحة
- **المحتوى:** شعار + 6 عناصر تنقل + محول اللغة
- **السلوكيات:**
  - Sticky عند التمرير (يتقلص الارتفاع)
  - قوائم منسدلة (Dropdown) لبعض العناصر
  - تمييز الصفحة النشطة (Active state)
  - خلفية شفافة في Hero → خلفية صلبة عند التمرير
- **المتغيرات:** Desktop (أفقي) | Mobile (Hamburger → Drawer)
- **الحالات:** Default | Scrolled | Mobile Open

### 1.2 Mobile Drawer — القائمة الجانبية
- **الوصف:** قائمة جانبية تنزلق من اليمين (RTL)
- **المحتوى:** جميع روابط التنقل + عناصر فرعية قابلة للتوسيع
- **السلوكيات:** فتح/إغلاق بحركة انزلاق + overlay خلفي

### 1.3 Breadcrumbs — مسار التنقل
- **الوصف:** مسار نصي يُظهر موقع الصفحة الحالية
- **المحتوى:** الرئيسية > القسم > الصفحة
- **الموقع:** أعلى المحتوى في جميع الصفحات الفرعية
- **الملاحظة:** فاصل "/" بين العناصر

### 1.4 Footer — التذييل
- **الوصف:** تذييل شامل يظهر في جميع الصفحات
- **المحتوى:**
  - شعار بترولات + وصف مختصر
  - 4-5 أعمدة روابط (عن الشركة، خدماتنا، فرص الأعمال، تواصل)
  - روابط وسائل التواصل الاجتماعي
  - حقوق النشر + سياسة الخصوصية
- **التخطيط:** متعدد الأعمدة (Desktop) | عمود واحد قابل للطي (Mobile)

## 2. مكونات Hero والعرض

### 2.1 Hero Section — القسم الرئيسي
- **الوصف:** قسم بارتفاع 80vh مع صورة خلفية ونص متراكب
- **المحتوى:** عنوان H1 + عنوان فرعي + (اختياري) زر CTA
- **التخطيط:** صورة عرض كامل + overlay تدرج داكن + نص على اليمين
- **المتغيرات:**
  - Hero رئيسي (الصفحة الرئيسية) — 80vh
  - Hero فرعي (صفحات داخلية) — 40-50vh
- **الملاحظات:** الصورة يجب أن تكون حقيقية لمحطة بترولات

### 2.2 Page Header — عنوان الصفحة الفرعية
- **الوصف:** شريط عنوان للصفحات الداخلية
- **المحتوى:** عنوان الصفحة + Breadcrumb + صورة خلفية
- **الارتفاع:** 300-400px
- **التخطيط:** خلفية صورة مع overlay + نص مركزي

## 3. مكونات المحتوى

### 3.1 Section Title — عنوان القسم
- **الوصف:** عنوان H2 + عنوان فرعي لكل قسم في الصفحة
- **المحتوى:** عنوان + وصف سطر أو سطرين
- **المحاذاة:** يمين (RTL) أو مركزي حسب السياق
- **الفاصل:** خط قصير ملون أسفل العنوان (4px عرض, 40-60px طول)

### 3.2 Stats Counter — عداد الإحصائيات
- **الوصف:** أرقام كبيرة مع أوصاف
- **المحتوى:** رقم + وصف (مثل: "+10 سنوات خبرة")
- **التخطيط:** 3-4 أعمدة أفقية
- **السلوك:** تأثير count-up عند الظهور في Viewport
- **الخلفية:** لون مميز أو خلفية داكنة

### 3.3 Timeline — الخط الزمني
- **الوصف:** عرض أحداث مرتبة زمنياً
- **المحتوى:** سنة + عنوان + وصف مختصر لكل حدث
- **التخطيط:** خط عمودي مع نقاط ومحتوى يتناوب يمين/يسار (Desktop) | عمود واحد (Mobile)

### 3.4 Rich Text — نص منسق
- **الوصف:** مكون لعرض المحتوى النصي الطويل (مقالات، أوصاف)
- **المحتوى:** عناوين، فقرات، قوائم، صور مدمجة
- **القواعد:** max-width: 720px | line-height: 1.7 | فقرات بمسافات واضحة

## 4. مكونات البطاقات

### 4.1 Service Card — بطاقة الخدمة
- **الوصف:** بطاقة تعرض خدمة واحدة
- **المحتوى:** أيقونة (48-64px) + عنوان + وصف (2-3 أسطر) + رابط "المزيد"
- **التخطيط:** عمودي (أيقونة أعلى)
- **الحالات:** Default | Hover (رفع خفيف + تمييز)
- **الاستخدام:** صفحة الخدمات، الصفحة الرئيسية

### 4.2 Project Card — بطاقة المشروع/المحطة
- **الوصف:** بطاقة تعرض محطة أو مشروع
- **المحتوى:** صورة + اسم المحطة + الموقع + قائمة خدمات متوفرة
- **التخطيط:** صورة أعلى + محتوى أسفل
- **الحالات:** Default | Hover
- **الاستخدام:** صفحة المحطات، الصفحة الرئيسية

### 4.3 News Card — بطاقة الخبر
- **الوصف:** بطاقة تعرض خبراً أو مقالاً
- **المحتوى:** صورة + تاريخ + تصنيف (Badge) + عنوان + مقتطف
- **التخطيط:** صورة أعلى + محتوى أسفل
- **الحالات:** Default | Hover
- **الاستخدام:** صفحة الأخبار، الصفحة الرئيسية

### 4.4 Team Card — بطاقة عضو الفريق
- **الوصف:** بطاقة تعرض عضو في فريق القيادة
- **المحتوى:** صورة شخصية + الاسم + المنصب + نبذة مختصرة
- **التخطيط:** صورة أعلى + نص أسفل | أو صورة يسار + نص يمين
- **الاستخدام:** صفحة القيادة

### 4.5 Job Card — بطاقة الوظيفة
- **الوصف:** بطاقة تعرض وظيفة متاحة
- **المحتوى:** عنوان الوظيفة + القسم + الموقع + نوع العمل + زر "قدم الآن"
- **التخطيط:** أفقي (سطر واحد)
- **الاستخدام:** صفحة التوظيف

## 5. مكونات التفاعل

### 5.1 CTA Section — قسم الدعوة للعمل
- **الوصف:** شريط عرض كامل يدعو للتواصل أو الإجراء
- **المحتوى:** عنوان + نص مختصر + زر أو زرين
- **التخطيط:** مركزي مع خلفية ملونة أو صورة
- **الاستخدام:** أسفل معظم الصفحات

### 5.2 Contact Form — نموذج الاتصال
- **الوصف:** نموذج لإرسال استفسار
- **الحقول:** الاسم (مطلوب) | البريد (مطلوب) | الهاتف | نوع الاستفسار (قائمة) | الرسالة (مطلوب)
- **التحقق:** تحقق فوري (Inline validation)
- **الحالات:** Default | Focus | Error | Success
- **Feedback:** رسالة نجاح بعد الإرسال

### 5.3 Franchise Form — نموذج طلب الفرنشايز
- **الوصف:** نموذج تفصيلي لطلب الفرنشايز
- **الحقول:** الاسم | البريد | الهاتف | المدينة | نوع الموقع | مساحة الأرض | رأس المال المتاح | رسالة إضافية
- **التخطيط:** متعدد الأعمدة | أو خطوات (Multi-step)

### 5.4 Button — زر
- **المتغيرات:**
  - Primary: خلفية ملونة + نص أبيض
  - Secondary: حدود + نص ملون
  - Ghost: نص فقط + سهم
- **الأحجام:** Large (48px) | Medium (40px) | Small (32px)
- **الحالات:** Default | Hover | Active | Disabled | Loading

## 6. مكونات العرض البصري

### 6.1 Partner Logos — شريط الشركاء
- **الوصف:** شريط أفقي يعرض شعارات الشركاء
- **المحتوى:** 5-8 شعارات
- **التخطيط:** صف أفقي مع محاذاة مركزية
- **النمط:** Grayscale بشكل افتراضي → ملون عند Hover
- **الاستخدام:** الصفحة الرئيسية، صفحة الشراكات

### 6.2 Image Gallery — معرض الصور
- **الوصف:** شبكة صور قابلة للتكبير
- **التخطيط:** شبكة 3-4 أعمدة
- **السلوك:** Lightbox عند النقر على الصورة
- **الاستخدام:** صفحة المعرض، صفحات المحطات

### 6.3 Map — الخريطة التفاعلية
- **الوصف:** خريطة Google Maps مع علامات المحطات
- **السلوك:**
  - علامات (Markers) لكل محطة
  - Popup عند النقر: اسم + عنوان + خدمات + رابط اتجاهات
  - فلتر بالمنطقة (اختياري)
- **الاستخدام:** صفحة المحطات، صفحة اتصل بنا

### 6.4 Feature Grid — شبكة المميزات
- **الوصف:** شبكة أيقونات مع أوصاف
- **المحتوى:** أيقونة + عنوان + وصف مختصر
- **التخطيط:** 2-3 أعمدة | أو 4 أعمدة للعناصر الصغيرة
- **الاستخدام:** صفحات الخدمات، الفرنشايز

## 7. مكونات مساعدة

### 7.1 Badge — شارة
- **الوصف:** علامة صغيرة للتصنيف
- **الاستخدام:** تصنيف الأخبار، حالة الوظيفة
- **الأحجام:** صغير (24px ارتفاع)

### 7.2 Divider — فاصل
- **الوصف:** خط أفقي رفيع للفصل بين المحتوى
- **النمط:** 1px solid | لون رمادي فاتح

### 7.3 Loading Skeleton — هيكل تحميل
- **الوصف:** عناصر رمادية متحركة أثناء تحميل المحتوى
- **الاستخدام:** بطاقات الأخبار، قائمة المحطات

### 7.4 Back to Top — العودة للأعلى
- **الوصف:** زر دائري صغير يظهر عند التمرير
- **الموقع:** أسفل يسار الشاشة (RTL)
- **السلوك:** يظهر بعد تمرير 300px | تمرير سلس للأعلى

### 7.5 Social Links — روابط التواصل
- **الوصف:** أيقونات وسائل التواصل الاجتماعي
- **المحتوى:** Facebook, X (Twitter), Instagram, YouTube, LinkedIn
- **الاستخدام:** Footer، صفحة اتصل بنا، مقالات الأخبار

### 7.6 Language Switcher — محول اللغة
- **الوصف:** زر تبديل بين العربية والإنجليزية
- **الموقع:** Navbar (أقصى اليسار)
- **النمط:** "AR | EN" نص بسيط
