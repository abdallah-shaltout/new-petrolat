# 13 — خارطة التطوير المستقبلية

## نظرة عامة

هذا المستند يحدد الميزات والأقسام التي يمكن إضافتها بعد إطلاق الموقع الأساسي. كل ميزة مُصنفة حسب الأولوية والتعقيد والأثر المتوقع.

---

## المرحلة 1: بعد الإطلاق (1-3 أشهر)

### 1.1 المدونة النشطة
- **الوصف:** قسم محتوى متجدد بمقالات متخصصة
- **الأقسام:**
  - نصائح صيانة السيارات
  - نصائح السفر على الطرق السريعة
  - أخبار قطاع الطاقة
  - أخبار الشركة وافتتاحاتها
- **الهدف:** SEO + بناء سلطة المحتوى + تفاعل الزوار
- **المتطلبات:** كاتب محتوى عربي، جدول نشر (2 مقالات/شهر)
- **التعقيد:** منخفض
- **الأولوية:** عالية

### 1.2 تطبيق خريطة المحطات المتقدم
- **الوصف:** خريطة تفاعلية متقدمة مع فلتر بالخدمات والمنطقة
- **الميزات:**
  - بحث بالمنطقة/المدينة
  - فلتر بالخدمات المتوفرة
  - اتجاهات مباشرة (Google Maps / Apple Maps)
  - عرض ساعات العمل الحية
- **الهدف:** تسهيل إيجاد المحطات
- **التعقيد:** متوسط
- **الأولوية:** عالية

### 1.3 نموذج التوظيف المتقدم
- **الوصف:** نظام تقديم وظائف متكامل
- **الميزات:**
  - رفع السيرة الذاتية
  - فلتر وظائف بالقسم والموقع
  - إشعارات وظائف جديدة
  - تتبع حالة الطلب (اختياري)
- **الهدف:** جذب الكفاءات بكفاءة
- **التعقيد:** متوسط
- **الأولوية:** متوسطة

---

## المرحلة 2: النمو (3-6 أشهر)

### 2.1 المركز الإعلامي الشامل (Media Center)
- **الوصف:** توسيع قسم الأخبار ليشمل:
  - نشرات صحفية
  - معرض صور مصنف
  - معرض فيديو
  - تقارير وإصدارات (Brochures, Annual Reports)
  - مواد إعلامية للتحميل (Media Kit)
- **الهدف:** دعم العلاقات الإعلامية والعامة
- **التعقيد:** متوسط
- **الأولوية:** متوسطة

### 2.2 برنامج الولاء الرقمي
- **الوصف:** صفحة تعريفية + ربط بتطبيق ولاء
- **الميزات:**
  - شرح البرنامج ومستوياته
  - طريقة الانضمام
  - المكافآت المتاحة
  - ربط بتطبيق الهاتف (إن وُجد)
- **الهدف:** ربط العملاء وزيادة الولاء
- **التعقيد:** متوسط (يعتمد على وجود نظام خلفي)
- **الأولوية:** متوسطة

### 2.3 الأسئلة الشائعة (FAQ)
- **الوصف:** قسم أسئلة شائعة منظم بالتصنيفات
- **التصنيفات:**
  - عام
  - الخدمات
  - الفرنشايز
  - التوظيف
  - الشكاوى والملاحظات
- **الهدف:** تقليل الاستفسارات المتكررة + SEO
- **التعقيد:** منخفض
- **الأولوية:** متوسطة

### 2.4 صفحة المسؤولية الاجتماعية المتقدمة
- **الوصف:** توسيع قسم CSR ليشمل:
  - مبادرات بيئية (كفاءة الطاقة، إدارة النفايات)
  - مساهمات مجتمعية
  - برامج السلامة المرورية
  - ربط برؤية 2030
- **الهدف:** تحسين صورة الشركة ودعم التموضع
- **التعقيد:** منخفض (محتوى أساساً)
- **الأولوية:** متوسطة

---

## المرحلة 3: التوسع (6-12 شهر)

### 3.1 بوابة المستثمرين (Investor Relations)
- **الوصف:** قسم مخصص للمستثمرين الحاليين والمحتملين
- **الأقسام:**
  - نظرة عامة استثمارية
  - التقارير المالية (إن كانت الشركة ستتحول لمساهمة)
  - العروض التقديمية
  - حوكمة الشركة
  - الأخبار المالية
- **الهدف:** جذب مستثمرين وبناء شفافية
- **التعقيد:** متوسط
- **الأولوية:** تعتمد على خطط الشركة

### 3.2 فرص الفرنشايز المتقدمة
- **الوصف:** توسيع قسم الفرنشايز ليشمل:
  - حاسبة عائد استثمار تقريبية
  - عرض تفاعلي لنموذج المحطة
  - خريطة المناطق المتاحة للفرنشايز
  - قصص نجاح شركاء الفرنشايز
  - نموذج طلب متعدد الخطوات
- **الهدف:** زيادة طلبات الفرنشايز
- **التعقيد:** عالي
- **الأولوية:** عالية (إذا كان الفرنشايز هدف نمو رئيسي)

### 3.3 بوابة الشركاء (Partner Portal)
- **الوصف:** منطقة محمية لشركاء الأعمال
- **الميزات:**
  - تسجيل دخول
  - مستندات وعقود
  - تقارير أداء
  - طلبات ومراسلات
- **الهدف:** تسهيل إدارة الشراكات
- **التعقيد:** عالي
- **الأولوية:** منخفضة (مرحلة لاحقة)

---

## المرحلة 4: الابتكار (12+ شهر)

### 4.1 محطات الشحن الكهربائي
- **الوصف:** قسم مخصص لخدمات شحن المركبات الكهربائية
- **المحتوى:**
  - أنواع الشواحن المتوفرة
  - مواقع محطات الشحن
  - خريطة شحن تفاعلية
  - حالة الشواحن (متاح/مشغول) — حي
- **الهدف:** مواكبة تحول الطاقة ورؤية 2030
- **التعقيد:** عالي (يحتاج تكامل IoT)
- **الأولوية:** استراتيجية — يعتمد على خطة الشركة

### 4.2 تطبيق الهاتف المحمول (Mobile App)
- **الوصف:** تطبيق مكمل للموقع
- **الميزات:**
  - محدد محطات
  - برنامج ولاء
  - طلب خدمات مسبق
  - إشعارات عروض
  - دفع إلكتروني
- **الهدف:** تجربة عميل متكاملة
- **التعقيد:** عالي جداً
- **الأولوية:** مرحلة لاحقة

### 4.3 لوحة تحكم (Dashboard) لإدارة المحتوى المتقدمة
- **الوصف:** لوحة تحكم مخصصة لفريق بترولات
- **الميزات:**
  - إدارة الأخبار والمدونة
  - إدارة الوظائف
  - إدارة طلبات الاتصال والفرنشايز
  - تحديث بيانات المحطات
  - تقارير إحصائية
- **الهدف:** سهولة إدارة المحتوى بدون مطور
- **التعقيد:** عالي
- **الأولوية:** عالية (تُبنى تدريجياً مع CMS)

```
  │              │                          │                      │                    │
  │              ├─ مدونة نشطة              ├─ مركز إعلامي         ├─ بوابة مستثمرين    ├─ شحن كهربائي

  │                                         └─ CSR متقدم

  └─ نماذج اتصال وفرنشايز
```

## معايير القرار

قبل تنفيذ أي ميزة مستقبلية، يجب تقييمها وفق:

| المعيار | السؤال |
|---------|--------|
| الأثر | هل ستزيد التحويلات أو حركة المرور بشكل ملموس؟ |
| التكلفة | هل تستحق الاستثمار مقارنة بالعائد؟ |
| التعقيد | هل يمكن بناؤها بموارد الفريق الحالية؟ |
| الصيانة | هل يمكن صيانتها وتحديثها بسهولة؟ |
| التوافق | هل تتوافق مع استراتيجية الشركة الحالية؟ |
| المستخدم | هل يطلبها المستخدمون فعلاً؟ |
