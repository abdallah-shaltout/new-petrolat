## 2. تفاصيل كل قسم

### الرئيسية (/)
- **الهدف:** انطباع أول قوي، توجيه الزائر حسب اهتمامه
- **الأقسام:**
  1. Hero Section — رسالة قوية + صورة محطة
  2. تعريف مختصر بالشركة
  3. الخدمات الرئيسية (بطاقات)
  4. أبرز المشاريع / المحطات
  5. أرقام وإحصائيات
  6. شركاء وعلامات تجارية
  7. آخر الأخبار
  8. CTA — تواصل / فرنشايز
- **المحتوى المطلوب:** نصوص تسويقية، صور محطات، أرقام حقيقية، شعارات شركاء

---

### عن بترولات (/about)

#### نبذة عن الشركة (/about/overview)
- **الهدف:** بناء الثقة والمصداقية
- **الأقسام:**
  1. قصة بترولات (التأسيس، النمو)
  2. خط زمني للإنجازات (Timeline)
  3. أرقام الشركة (محطات، موظفين، مناطق)
  4. القيم المؤسسية
- **المحتوى المطلوب:** نص سردي، تواريخ، أرقام، صور أرشيفية

#### رؤيتنا ورسالتنا (/about/vision)
- **الهدف:** عرض التوجه الاستراتيجي
- **الأقسام:**
  1. الرؤية
  2. الرسالة
  3. الأهداف الاستراتيجية
  4. الربط برؤية 2030
- **المحتوى المطلوب:** نصوص استراتيجية، أيقونات توضيحية

#### القيادة (/about/leadership)
- **الهدف:** إظهار الكفاءات البشرية
- **الأقسام:**
  1. مجلس الإدارة / الإدارة العليا
  2. صور ونبذ شخصية
- **المحتوى المطلوب:** صور رسمية، سير ذاتية مختصرة

#### الجودة والسلامة (/about/quality)
- **الهدف:** إثبات الالتزام بالمعايير
- **الأقسام:**
  1. الشهادات والاعتمادات
  2. معايير السلامة المعتمدة
  3. إجراءات الجودة
  4. التدريب والتأهيل
- **المحتوى المطلوب:** صور شهادات، وصف إجراءات، أيقونات معايير

#### المسؤولية الاجتماعية (/about/csr)
- **الهدف:** إظهار الالتزام المجتمعي
- **الأقسام:**
  1. مبادرات بيئية
  2. مساهمات مجتمعية
  3. السلامة المرورية
- **المحتوى المطلوب:** نصوص، صور فعاليات

---

### خدماتنا (/services)

#### صفحة الخدمات الرئيسية
- **الهدف:** عرض شامل لجميع الخدمات
- **الأقسام:**
  1. مقدمة عامة عن الخدمات
  2. بطاقات الخدمات (7 خدمات) مع روابط للتفاصيل
  3. CTA — اتصل للمزيد
- **كل بطاقة تحتوي:** أيقونة، عنوان، وصف مختصر، رابط "المزيد"

#### صفحة خدمة مفردة (نموذج موحد)
- **الهدف:** شرح تفصيلي للخدمة
- **الأقسام:**
  1. عنوان الخدمة + صورة رئيسية
  2. وصف الخدمة
  3. المميزات / الفوائد
  4. آلية العمل / الخطوات
  5. القطاعات المستفيدة
  6. CTA — اتصل / زر محطة
- **المحتوى المطلوب:** نص تفصيلي، صور حقيقية، أيقونات

---

### مشاريعنا (/projects)

#### محطاتنا (/projects/stations)
- **الهدف:** عرض شبكة المحطات
- **الأقسام:**
  1. خريطة تفاعلية (Google Maps embedded)
  2. قائمة المحطات مع فلتر بالمنطقة
  3. كل محطة: اسم، عنوان، خدمات متوفرة، ساعات العمل
  4. صور المحطة
- **المحتوى المطلوب:** إحداثيات GPS، صور، تفاصيل خدمات كل محطة

#### مشاريع تحت التطوير (/projects/upcoming)
- **الهدف:** إظهار النمو والتوسع
- **الأقسام:**
  1. مشاريع جديدة قيد الإنشاء
  2. خطط التوسع الجغرافي
  3. جدول زمني تقريبي
- **المحتوى المطلوب:** خرائط، تواريخ، أوصاف مشاريع

---

### المركز الإعلامي (/media)

#### الأخبار (/media/news)
- **الهدف:** محتوى متجدد + SEO
- **الأقسام:**
  1. قائمة أخبار مع تصنيف
  2. صفحة خبر مفردة
  3. فلتر بالتاريخ والتصنيف
- **المحتوى المطلوب:** مقالات دورية (2-4 شهرياً)

#### المعرض (/media/gallery)
- **الهدف:** عرض بصري للمحطات والمشاريع
- **الأقسام:**
  1. ألبومات مصنفة (محطات، فعاليات، مرافق)
  2. عرض صور بمعرض تفاعلي
- **المحتوى المطلوب:** صور عالية الجودة

---

### فرص الأعمال (/business)

#### الفرنشايز (/business/franchise)
- **الهدف:** جذب مستثمرين للفرنشايز
- **الأقسام:**
  1. لماذا فرنشايز بترولات؟
  2. نموذج العمل
  3. المتطلبات
  4. خطوات التقديم
  5. قصص نجاح (إن وجدت)
  6. نموذج طلب فرنشايز
- **المحتوى المطلوب:** نصوص تسويقية، أرقام عائد، نموذج إلكتروني

#### الشراكات (/business/partnerships)
- **الهدف:** جذب شركاء تجاريين
- **الأقسام:**
  1. أنواع الشراكات المتاحة
  2. الشركاء الحاليون
  3. نموذج تواصل للشراكات
- **المحتوى المطلوب:** شعارات شركاء، أنواع التعاون

#### التوظيف (/business/careers)
- **الهدف:** جذب الكفاءات
- **الأقسام:**
  1. لماذا بترولات؟ (ثقافة العمل)
  2. الوظائف المتاحة
  3. نموذج تقديم
- **المحتوى المطلوب:** نصوص، صور بيئة عمل، وصف وظائف

---

### اتصل بنا (/contact)
- **الهدف:** تسهيل التواصل
- **الأقسام:**
  1. معلومات الاتصال (هاتف، بريد، عنوان)
  2. خريطة المقر الرئيسي
  3. نموذج اتصال (نوع الاستفسار: عام / فرنشايز / شراكة / توظيف)
  4. روابط وسائل التواصل
- **المحتوى المطلوب:** بيانات اتصال، خريطة

## 5. قواعد التنقل

| القاعدة | التفاصيل |
|---------|----------|
| عمق التنقل | حد أقصى 3 مستويات |
| عناصر Navbar | 6 عناصر رئيسية + لغة |
| القوائم المنسدلة | فقط لـ"عن بترولات" و"خدماتنا" و"المركز الإعلامي" و"فرص الأعمال" |
| CTA في Navbar | لا يوجد زر CTA في الشريط العلوي (يبقى نظيفاً) |
| Breadcrumbs | في جميع الصفحات الفرعية |
| Active state | تمييز الصفحة الحالية في التنقل |
| Mobile menu | قائمة جانبية (Drawer) مع جميع الروابط |
| Sticky navbar | نعم — يثبت عند التمرير مع تقليص الحجم |
