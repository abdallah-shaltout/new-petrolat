## 4. البيانات المنظمة (Schema Markup)

### Schema المطلوبة

#### Organization (على جميع الصفحات)
```json
{
  "@type": "Organization",
  "name": "بترولات",
  "alternateName": "Petrolat",
  "url": "https://www.petrolat.com",
  "logo": "https://www.petrolat.com/logo.png",
  "contactPoint": {
    "@type": "ContactPoint",
    "telephone": "+966112961816",
    "contactType": "customer service",
    "availableLanguage": ["Arabic", "English"]
  },
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "الرياض",
    "addressRegion": "منطقة الرياض",
    "addressCountry": "SA"
  },
  "sameAs": [
    "https://facebook.com/petrolatstation",
    "https://instagram.com/petrolatstation",
    "https://twitter.com/petrolatstation"
  ]
}
```

#### GasStation (لكل محطة)
```json
{
  "@type": "GasStation",
  "name": "محطة بترولات — [اسم الموقع]",
  "address": { ... },
  "geo": { "latitude": "...", "longitude": "..." },
  "openingHoursSpecification": { "24/7" },
  "hasOfferCatalog": { "fuel types..." }
}
```

#### BreadcrumbList (على جميع الصفحات الفرعية)
```json
{
  "@type": "BreadcrumbList",
  "itemListElement": [...]
}
```

#### FAQPage (صفحة الفرنشايز)
```json
{
  "@type": "FAQPage",
  "mainEntity": [...]
}
```

#### JobPosting (صفحة التوظيف)
```json
{
  "@type": "JobPosting",
  "title": "...",
  "datePosted": "...",
  "employmentType": "..."
}
```

## 6. SEO التقني (Technical SEO)

### المتطلبات الأساسية

| العنصر | المواصفة |
|--------|---------|
| SSL/HTTPS | إلزامي على جميع الصفحات |
| Sitemap XML | تلقائي مع Nuxt، يشمل جميع الصفحات |
| Robots.txt | السماح بفهرسة جميع الصفحات العامة |
| Canonical URLs | على جميع الصفحات (خاصة العربي/الإنجليزي) |
| Hreflang | ar-SA للعربية، en للإنجليزية |
| Page Speed | LCP < 2.5s, FID < 100ms, CLS < 0.1 |
| Mobile-First | تصميم متجاوب بالكامل |
| Open Graph | عنوان + وصف + صورة لكل صفحة |
| Twitter Cards | summary_large_image |
| Favicon | متعدد الأحجام |
| 404 Page | صفحة مخصصة مع روابط مفيدة |
| URL Structure | نظيفة، وصفية، بالإنجليزية |

### هيكل URLs

```
✅ /services/fuel
✅ /about/quality
✅ /media/news/station-opening-riyadh

❌ /page?id=123
❌ /services/1
❌ /خدمات/الوقود (تجنب العربية في URLs)
```

### الربط الداخلي (Internal Linking)
- كل صفحة خدمة تربط بصفحات خدمات ذات صلة
- كل مقال مدونة يربط بصفحة خدمة مناسبة
- صفحة المحطات تربط بصفحات الخدمات
- Footer يحتوي روابط لجميع الأقسام الرئيسية
- Breadcrumbs في جميع الصفحات الفرعية

### أدوات التتبع المطلوبة
1. **Google Analytics 4** — تتبع الزيارات والتحويلات
2. **Google Search Console** — مراقبة الفهرسة والأداء
3. **Google Tag Manager** — إدارة أكواد التتبع
4. **تتبع أهداف التحويل:**
   - إرسال نموذج فرنشايز
   - إرسال نموذج اتصال
   - النقر على رقم الهاتف
   - النقر على البريد الإلكتروني
   - تحميل كتيب الشركة
   - فتح اتجاهات Google Maps

## 7. SEO المحلي (Local SEO)

### Google Business Profile
- إنشاء/تحديث ملف Google Business لكل محطة
- فئة رئيسية: Gas Station
- فئات ثانوية: Auto Repair Shop, Car Wash, Convenience Store
- صور محدثة لكل محطة
- ساعات عمل دقيقة
- ربط بالموقع الإلكتروني

### Google Maps
- تسجيل جميع المحطات على Google Maps
- التأكد من دقة الإحداثيات
- إضافة صور ومعلومات كاملة
- تشجيع التقييمات من العملاء
